# 小红书封面 + 标题 + 正文生成器

基于爆款规律，为用户输入的内容自动生成高点击率的封面布局方案、多版本标题和小红书正文内容。

## 核心规律库

### 标题公式

| 公式类型 | 模板 | 适用场景 |
|----------|------|----------|
| **数字钩子** | `[N]个[工具/技巧]，[利益点]` | 工具清单、干货分享 |
| **对比型** | `[A] vs [B]，有什么区别？` | 概念解读、测评 |
| **反常识** | `[反常识观点]？[真相揭露]` | 认知颠覆、争议话题 |
| **低门槛** | `[0基础/小白][动词][成果]` | 教程、入门指南 |
| **情绪爆破** | `[动词][痛点]，这招太绝了` | 技巧分享、种草 |
| **钩子+金句** | `[年龄/身份][成就]｜[金句]` | 人设故事、经验分享 |

### 封面视觉规律

#### 三层信息架构

```
┌────────────────────────────────┐
│  [顶部标签] 定位标签/分类       │ ← 品牌统一色（红/蓝/黑）
│                                │
│  [中部核心] 大字主标题          │ ← 高饱和撞色（荧光绿/亮黄/紫蓝）
│            + 人物/工具图标      │
│                                │
│  [底部钩子] 副标题/关键词组     │ ← 白字+描边 或 小标签矩阵
└────────────────────────────────┘
```

#### 高转化颜色组合

| 配色方案 | 情绪传达 | 适用类型 |
|----------|----------|----------|
| 亮黄 + 黑底 | 搞钱、冲击 | 副业、赚钱、效率 |
| 荧光绿 + 深色 | 科技、抓眼 | AI工具、技术教程 |
| 红 + 白 + 黑 | 专业、严肃 | 职场、知识干货 |
| 暖光 + 柔色字 | 亲和、信任 | 人设故事、生活分享 |

#### 必备视觉元素

1. **真人出镜** — 信任感加成
2. **情绪化表情** — 惊讶/震撼/发现宝藏 > 标准职业照
3. **工具Logo矩阵** — 直接放App图标在画面（用户辨识度高）
4. **指向性图形** — 箭头、勾选、高亮框
5. **关键词环绕** — 在空白处散布痛点词/场景词

## 正文写作规律

### 正文结构公式

小红书爆款正文遵循「钩子→价值→行动」三段式：

```
【开头钩子】1-2句，制造共鸣或悬念
   ↓
【核心价值】干货内容，分点呈现
   ↓
【结尾行动】引导互动或私信
```

### 正文写作技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| **emoji分隔** | 每段用emoji开头，增加可读性 | 📌 第一点... 💡 第二点... |
| **短句为主** | 每句不超过20字，避免长段落 | ❌ 长句子 → ✅ 短句。换行。再短句。 |
| **口语化** | 像和朋友聊天，避免书面语 | ❌「因此」→ ✅「所以」 |
| **数字具象** | 用具体数字代替模糊描述 | ❌「很快」→ ✅「10分钟」 |
| **痛点共鸣** | 开头直击用户痛点 | 「是不是也遇到过...」 |
| **金句收尾** | 结尾留下记忆点 | 「2026年，AI是用来干活的，不是用来聊天的」 |
